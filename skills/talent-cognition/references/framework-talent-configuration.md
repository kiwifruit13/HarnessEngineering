## 概览

本文档为12个Prompt框架提供完整的天赋配置方案。每个方案都经过系统化的分析和优化，确保框架具备"天赋型专家"的能力。

**配置方案特点**：
- 系统化：使用天赋配置算法进行分析
- 优化：经过多轮测试和优化
- 可用：配置方案可直接使用
- 可调：可根据场景调整参数

**使用说明**：
1. 根据框架类型选择对应的配置方案
2. 根据具体场景调整参数
3. 根据反馈持续优化配置

## 极简型框架

### A.P.E框架

**框架类型**：极简型
**核心价值**：快速澄清问题，定义行动、目的和期望
**适用场景**：概念解释、快速诊断、问题澄清

#### 天赋配置方案

**核心天赋（必需）**：
1. **快速模式识别**（强度：8/10，速度：快）
   - 作用：在10秒内识别问题核心
   - 应用：ACTION要素

2. **本质洞察**（强度：7/10，速度：中）
   - 作用：理解用户真正意图
   - 应用：PURPOSE要素

3. **结果导向**（强度：8/10，速度：快）
   - 作用：以结果为核心
   - 应用：EXPECTATION要素

4. **清晰表达**（强度：9/10，速度：快）
   - 作用：清晰简洁地表达
   - 应用：贯穿全框架

**辅助天赋（可选）**：
- **简洁表达**（强度：6/10）：增强简洁性
- **抽象概括**（强度：6/10）：支持ACTION要素

**排除天赋**：
- 发散思维（与快速识别冲突）
- 细节感知（与简洁表达冲突）

#### 激活顺序

```
1. 快速模式识别（10秒）→ 识别问题核心
2. 本质洞察（10秒）→ 理解真正意图
3. 抽象概括（5秒）→ 概括核心
4. 结果导向（10秒）→ 定义期望结果
5. 清晰表达 + 简洁表达（10秒）→ 输出
```

#### 执行指令

```markdown
当使用A.P.E框架时：
1. 立即激活快速模式识别，10秒内锁定问题核心
2. 紧接着激活本质洞察，理解用户真正意图
3. 使用抽象概括，将核心和意图概括为简洁表达
4. 激活结果导向，定义清晰的期望结果
5. 使用清晰表达和简洁表达，输出简洁准确的回答

输出要求：
- ACTION：1句话概括
- PURPOSE：1句话说明
- EXPECTATION：1-3个具体点
- 总体：10-30秒阅读即可理解
```

#### 配置评分

- 响应速度：9/10（平均20秒）
- 输出质量：9/10（准确率90%）
- 资源消耗：9/10（消耗低）
- 综合评分：8.875/10
