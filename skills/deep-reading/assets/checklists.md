# 批判性阅读检查清单

## 论文批判性阅读检查清单

### 研究设计维度
使用此清单评估论文的研究设计是否严谨。

- [ ] **研究问题清晰**
  - 研究问题是否明确表述？
  - 研究目标是否具体可测？

- [ ] **研究缺口识别**
  - 是否说明了现有研究的不足？
  - 本研究填补了什么缺口？

- [ ] **样本代表性**
  - 样本量是否充足？
  - 样本选择是否有偏差？
  - 能否推广到更大群体？

- [ ] **对照组设置**
  - 是否设置了合适的对照组？
  - 控制变量是否充分考虑？

- [ ] **实验条件可复现**
  - 实验设计是否详细描述？
  - 关键参数是否明确？
  - 其他研究者能否复现？

- [ ] **变量控制**
  - 是否识别了所有相关变量？
  - 控制变量是否有效控制？
  - 是否存在混淆变量？

---

### 数据分析维度
使用此清单评估数据分析是否规范和透明。

- [ ] **统计方法适合**
  - 统计方法是否适合数据类型？
  - 是否说明了选择该方法的原因？

- [ ] **p值正确解读**
  - 是否过度解读统计显著性？
  - 是否报告了效应量？
  - 是否考虑了置信区间？

- [ ] **图表完整呈现**
  - 图表是否完整呈现所有数据？
  - 是否选择性展示"好看"的结果？
  - 图表标注是否清晰？

- [ ] **异常值处理**
  - 异常值如何识别？
  - 排除标准是否明确？
  - 是否报告了异常值的影响？

- [ ] **多重比较校正**
  - 是否进行了多重比较？
  - 是否进行了校正（如Bonferroni）？
  - 结果是否经过校正？

- [ ] **数据透明**
  - 原始数据是否公开？
  - 数据是否经过预处理？如何预处理？

---

### 论证逻辑维度
使用此清单评估论证过程是否严密。

- [ ] **结论自然得出**
  - 结论是否从数据中自然得出？
  - 是否存在过度推断？
  - 结论是否与研究问题匹配？

- [ ] **替代解释讨论**
  - 是否考虑了其他可能的解释？
  - 是否解释了为什么排除这些解释？
  - 是否选择性无视矛盾证据？

- [ ] **因果推断谨慎**
  - 相关性是否被误认为是因果性？
  - 是否讨论了因果关系的必要条件？
  - 是否有实验设计支持因果推断？

- [ ] **局限性坦诚**
  - 作者是否承认研究的局限性？
  - 局限性是否具体明确？
  - 是否试图掩盖缺陷？

- [ ] **外部效度讨论**
  - 结果能否推广到其他场景？
  - 边界条件是什么？
  - 适用范围是否明确？

---

### 可复现性维度
使用此清单评估研究的透明度和可复现性。

- [ ] **方法描述详细**
  - 方法部分是否足够详细？
  - 其他研究者能否据此复现？
  - 关键细节是否遗漏？

- [ ] **数据公开**
  - 数据是否公开？
  - 数据获取方式是否说明？
  - 数据格式是否标准化？

- [ ] **代码公开**
  - 分析代码是否公开？
  - 代码是否有文档说明？
  - 代码是否能运行？

- [ ] **预注册**
  - 研究是否预注册？
  - 是否存在p-hacking迹象？
  - 数据采集是否规范？

## 逻辑谬误检测清单

### 常见逻辑谬误

- [ ] **以偏概全（Hasty Generalization）**
  - 基于有限样本得出普遍结论
  - 示例："我认识三个程序员都很内向，所以程序员都内向"

- [ ] **因果倒置（Reverse Causation）**
  - 混淆因果方向
  - 示例："成功的公司都喝了咖啡，所以喝咖啡导致成功"

- [ ] **稻草人攻击（Straw Man）**
  - 扭曲对方观点后攻击
  - 示例："他说X，但他其实是说Y（扭曲后的Y）"

- [ ] **滑坡谬误（Slippery Slope）**
  - 毫无根据的连锁推论
  - 示例："如果我们允许A，就会导致B，然后C，最终灾难"

- [ ] **错误归因（False Cause）**
  - 将相关性误认为因果性
  - 示例："冰淇淋销量增加时，溺水事故增加，所以冰淇淋导致溺水"

- [ ] **诉诸权威（Appeal to Authority）**
  - 仅依赖权威而非证据
  - 示例："这是专家说的，所以一定对"

- [ ] **人身攻击（Ad Hominem）**
  - 攻击对方而非论证
  - 示例："他是傻瓜，所以他的观点是错的"

- [ ] **虚假两难（False Dilemma）**
  - 强制二选一，忽略其他选项
  - 示例："要么完全支持，要么完全反对"

### 批判性提问技巧

针对每个论点，提出以下问题：

1. **论点是什么？**
   - 明确作者试图证明的结论

2. **论据是什么？**
   - 识别支撑论点的证据

3. **论据是否充分？**
   - 证据数量是否足够？
   - 证据质量是否可靠？

4. **是否有替代解释？**
   - 还有什么其他可能性？
   - 作者是否排除了其他可能性？

5. **论证是否有漏洞？**
   - 是否存在逻辑跳跃？
   - 是否存在未被解决的矛盾？

6. **证据是否被曲解？**
   - 证据是否被选择性呈现？
   - 数据是否被误导性解读？
